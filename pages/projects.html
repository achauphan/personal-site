repository

Description:

Simple imagine color filtering by inputting an image and a set number of colors to filter. Uses a k-means clustering algorithm to find organize pixels to the cluster that is closes to the pixel's color. The color of each cluster is then re-evaluated to the average color of all the pixels which belong to that cluster and then each pixel's color is then assigned to their corresponding cluster's new color. This process continues until there is little to no changes in cluster color.

Python
